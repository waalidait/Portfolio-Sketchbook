# 📓 Walid Ait Tazount - Scrapbook Portfolio

Welcome to my personal portfolio! This project features a unique **Scrapbook** and **Brutalist** design, blending the raw aesthetic of a physical diary with professional web development. 

## 🚀 Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom animations, responsive layouts, and Dark Mode implementation.
- **JavaScript (Vanilla):** Scroll animations using Intersection Observer and Dark Mode logic.
- **FontAwesome:** For all stylized icons.

## ✨ Key Features
- **Unique UI:** A layered "paper and tape" design inspired by scrapbook aesthetics.
- **Dynamic Dark Mode:** A full theme switch that preserves the high-contrast design.
- **Interactive Custom Cursor:** A stylized pointer that matches the site's retro/manual vibe.
- **Interactive Navigation:** A "Get In Touch" button with physical press effects and guided arrows.
